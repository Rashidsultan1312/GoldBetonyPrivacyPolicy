Privacy Policy for Gold Betony

Last updated: April 06, 2026

This Privacy Policy describes what data may be processed when using the Gold Betony application, the purposes for which such data is used, and the rights available to users.

Gold Betony is an informational sports application for a broad audience interested in football, match statistics, teams, and analytical indicators obtained from third-party data sources.
General Provisions
We respect users’ privacy and aim to process only the amount of data necessary for the proper operation of the application, the display of statistics, the improvement of service stability, and the protection of the service.

By using Gold Betony, the user agrees to this Privacy Policy.
Technical Data
When using the application, technical data necessary for its operation may be processed automatically, including:

- IP address
- device type and model
- operating system version
- application version
- device language
- technical and diagnostic data
- information about network requests required to load statistics
- information about errors and crashes

This data is used only to the extent necessary to provide users with current information, ensure the proper operation of the application, diagnose technical issues, and improve service quality.
In-App Activity Data
The application may process anonymized information about user activity within the application, including:

- viewed sections
- selected teams, matches, or statistical screens
- technical interface events

Such information is used to improve the structure, performance, and usability of the application.
Data Not Manually Provided by the User
The application does not require users to provide registration data in order to access its main features. We do not use a user account system and do not collect user registration data unless otherwise expressly stated within the application itself.
What the Application Does Not Do
At this time, the application:

- does not require account creation
- does not use email marketing or promotional mailings
- does not include public profiles, comments, chats, or other social features
- does not sell users’ personal data
- does not use user data for independent third-party marketing purposes
Use of Third-Party Data Sources
The application uses third-party data sources to display statistical and analytical information.

When the application connects to external data providers, the information necessary to respond to a request may be technically processed, including:

- IP address
- request parameters
- time of request
- technical information about the device and application
- server logs and diagnostic data

Such processing is carried out only to the extent necessary to provide users with statistics and ensure the functionality of the service.

We do not operate our own user account system and do not store users’ personal data on our own servers, except where this is expressly required for a specific application feature or expressly stated within the application.
Profile Photo, Camera, and Photo Library
The application may request access to the device camera or photo library only for features related to selecting or creating a profile image.

If the user grants such permission:

- images and related data are used locally on the user’s device
- such images are not automatically uploaded to our servers
- the user independently manages permissions through the device settings

If access to the camera or photo library is not granted, the application will continue to function within the limits of the available functionality.
Data Retention
We aim to minimize the amount of data processed and stored.

We do not store users’ personal data on our own servers unless this is required for a specific application feature.

Certain application settings, user preferences, and local interface elements may be stored on the user’s device until the application is deleted or the application data is cleared by the user.

Because the application uses external data sources, technical request logs and other service-related data may be processed and stored by the relevant data providers in accordance with their internal rules, technical infrastructure, and applicable legal requirements.
Disclosure to Third Parties
Data may be disclosed to third parties only in the following cases:

- where necessary to provide statistics and operate the application through external data providers
- where required for technical support, hosting, or infrastructure maintenance
- where such disclosure is required by law or by a lawful request from competent authorities
- where necessary to protect the rights, safety, and stability of the application

We do not disclose user data to third parties for their own independent marketing purposes.
International Data Processing
Because the application uses external data sources and third-party technical infrastructure, certain technical data may be processed on servers located outside the user’s country.

In such cases, we seek to use reasonable and appropriate measures to protect data, taking into account the nature of the service and the technical infrastructure used.
User Rights
Users may:

- submit a request regarding data processing
- request deletion of data if such data is actually stored by us and makes it possible to identify the user
- disable the application’s access to the camera and photo library through device settings
- stop using the application and delete it from their device

If certain technical data is processed by third-party data providers, the ability to fulfill specific requests may depend on the nature of such processing and the technical ability to identify a specific user.
Children
Gold Betony is not specifically directed to children. It is intended for a broad audience interested in football, sports statistics, teams, matches, and analytical information.

We do not knowingly seek to collect personal data from children. If we become aware that such data has been provided without the required consent, we will take reasonable steps to delete it within the limits of our technical capabilities.
Data Security
We take reasonable organizational and technical measures to protect data against unauthorized access, alteration, disclosure, or destruction.

However, no method of data transmission over the Internet and no method of electronic storage can guarantee absolute security.
Changes to This Privacy Policy
We may update this Privacy Policy from time to time. The current version of the policy will be published together with the date of the latest update.

Continued use of the application after the publication of an updated version of the policy means that the user accepts its current version.
